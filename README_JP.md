# 大型マート顧客RFM分析プロジェクト

過去2年分の購買データを使い**RFM分析**で顧客をセグメント化し、売上に最も貢献するコア顧客層を見つけ、マーケティング戦略を立案しました。

## 主な内容
- **データ**: 2年間の顧客購買履歴  
- **手法**: RFM（Recency, Frequency, Monetary）  
  - Recency: 最終購入からの経過日数  
  - Frequency: 購入回数  
  - Monetary: 総購入金額  
- **プロセス**  
  1. データ探索と前処理  
  2. RFM指標の算出  
  3. スコアリングと顧客セグメンテーション  
  4. セグメント別売上寄与度と特性分析  
- **目的**: 限られたマーケティング予算で最大の効果を出すコア顧客ターゲティング

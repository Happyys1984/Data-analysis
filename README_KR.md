# 대형 마트 고객 RFM 분석 프로젝트

2년 치 고객 구매 데이터를 가지고 **RFM 분석**을 통해 고객을 세분화하고, 매출에 가장 크게 기여하는 핵심 고객층을 찾아 마케팅 전략을 세워보았습니다.

## 주요 내용
- **데이터**: 2년간 고객 구매 내역
- **방법**: RFM (Recency, Frequency, Monetary)
  - Recency: 최근 구매일
  - Frequency: 구매 빈도
  - Monetary: 총 구매 금액
- **과정**
  1. 데이터 탐색 및 전처리
  2. RFM 지표 계산
  3. 점수화 후 고객 군집화
  4. 군집별 매출 기여도 및 특성 분석
- **목표**: 한정된 마케팅 자원으로 최대 효과 내는 핵심 고객 타겟팅
